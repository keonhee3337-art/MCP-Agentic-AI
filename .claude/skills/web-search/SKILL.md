# Skill: Web Search

Search the web directly from Claude Code — no manual tab-switching to Gemini or a browser.

## Trigger phrases
- "search for X"
- "look up X"
- "find recent info on X"
- "what's the latest on X"

## How to use
Just ask naturally. Examples:
- "Search for the best LangChain tutorials for beginners"
- "Look up what Kearney does in AI consulting"
- "Find recent news on agentic AI frameworks"

## What Claude does
1. Uses the WebSearch tool to query the web
2. Returns a bullet-point summary of the most relevant results
3. Includes source links for anything you might want to dig into

## Output format
- Bullet points — no paragraphs
- Lead with the most useful finding
- List sources at the bottom
- Flag if results are outdated or thin

## Notes
- Use this instead of switching to Gemini for quick lookups
- For deep research, say "deep search on X" and Claude will run multiple queries and synthesize
